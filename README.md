# Desafio-N3-Laboratorio1
Ejercicio práctico de Desarrollo Web: continuación del archivo de “Laboratorio 2”, trabajando todos los textos dentro de un div y aplicando propiedades CSS de tipografía, color y fondo. Se destaca el uso de strong en el 50% para resaltar descuentos y la correcta implementación de variantes tipográficas con Google Fonts.
